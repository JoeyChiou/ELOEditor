Source Code

All the source code necessary to publish the electronic files presented in this content collection is included as a resource. For each Plug-In Technology Example, the associated source files are included in the respective directories. These files are also listed as assets for each technology in the resources section of the manifest.

To view the source files included, you will need to decompress this content package.

  • $CONTENT_ROOT/Source Code Instructions/Source Code/pite.html
  • $CONTENT_ROOT/Shared Files/scripts/APIWrapper.js
